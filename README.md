## College Event Management Website

## Project Overview

This project is a multi-page College Event Management Website developed using Semantic HTML5 and Accessibility Standards. The website allows users to explore events, view schedules, register for activities, and contact event organizers through accessible forms.

## Features

* Semantic HTML5 Structure
* Multi-Page Website Architecture
* Accessible Navigation Menu
* ARIA Labels for Screen Readers
* SEO-Friendly Meta Tags
* Event Listings and Information
* Event Schedule Page
* Student Registration Form
* Contact Form
* Keyboard Navigable Interface
* WCAG Accessibility Best Practices

## Pages Included

### Home

Provides an introduction to the College Event Management Website and highlights featured events.

### Events

Displays technical, cultural, and sports events available for students.

### Schedule

Shows the timetable and event schedule.

### Registration

Allows students to register for events using an accessible form.

### Contact

Provides a contact form for inquiries and communication with organizers.
